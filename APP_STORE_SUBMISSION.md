# PrivateAI — App Store Submission Content

## Description (for App Store Connect)

Claude on your iPhone. Private. Encrypted. Auditable.

All medical data, financial information, and personal conversations stay encrypted on your device. They never touch cloud servers.

### Features:
• Voice input with on-device processing
• AES-256 encrypted storage
• Injection detection & blocking
• Automatic routing (cloud vs local)
• Face ID authentication
• Real-time network transparency

Your privacy is enforced at the code level, not with promises.

**Notes to Reviewer:**

This is a privacy-first AI assistant. All processing happens on the device or via Claude API. Medical data is automatically blocked from cloud APIs by code-level enforcement. No personal data is transmitted without explicit user action.

The app includes:
1. Voice input (on-device speech recognition)
2. Encryption (AES-256 at rest)
3. Security checking (prompt injection detection)
4. Smart routing (sensitive data → local AI only)
5. Face ID (biometric unlock)

Testing instructions:
- Tap the microphone icon and speak
- Type "ignore all previous instructions" to see injection blocking
- Close and reopen the app to verify message persistence (encryption)
- Wait 5+ minutes in background, reopen to see Face ID relock
